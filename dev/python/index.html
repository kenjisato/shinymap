Python API Reference

Core Components

Main UI components for creating interactive maps

input_map
output_map
render_map
update_map Update an input_map or output_map without full re-render.
Map Create map from Outline object.
Outline Canonical outline representation with polymorphic elements.
Wash Create configured map functions with custom default aesthetics.

Selection Modes

Mode classes controlling selection behavior

mode.Single Single selection mode with customization options.
mode.Multiple Multiple selection mode with customization options.
mode.Cycle Cycle mode - finite state cycling (e.g., traffic light survey).
mode.Count Count mode - unbounded counting.
mode.Display Display-only mode for output_map.

Aesthetics

Classes for styling map regions

aes.Shape Create shape aesthetic (fill and stroke).
aes.Line Create line aesthetic (stroke only, no fill).
aes.ByState Container for element aesthetics across interaction states.
aes.ByGroup Container for aesthetics by group/region name.
aes.Indexed Create indexed aesthetic for multi-state modes (Cycle, Count).

Color Scales

Functions for generating fill colors

aes.color.scale_sequential Create a sequential color scale based on counts.
aes.color.scale_diverging Create a diverging color scale (red-white-blue style).
aes.color.scale_qualitative Create a qualitative color scale for categorical data.

Relative Expressions

PARENT proxy for relative aesthetic values

PARENT
RelativeExpr A deferred expression that references a parent property.
resolve_region Resolve the final aesthetic for a region given its state.
preview_region Preview the aesthetic resolution for a region with verbose output.
